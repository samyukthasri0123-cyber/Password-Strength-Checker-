Advanced Password Strength Checker

Description
This project is an Advanced Password Strength Checker developed using Python. It analyzes the strength of a password based on multiple security criteria such as password length, uppercase letters, lowercase letters, digits, special characters, repeated characters, sequential patterns, common passwords, and entropy calculation.

Features
- Password Length Validation
- Uppercase & Lowercase Detection
- Number Detection
- Special Character Detection
- Common Password Detection
- Sequential Pattern Detection
- Repeated Character Detection
- Password Entropy Calculation
- Security Score (0–100)
- Strength Classification
- Security Suggestions

Technologies Used
- Python
- Object-Oriented Programming (OOP)
- Regular Expressions (`re`)
- String Module
- Math Module
- Collections Module
- Datetime Module

How to Run

bash
python password_checker.py
